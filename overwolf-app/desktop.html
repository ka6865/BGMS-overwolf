BGMS Companion for PUBG

Live match context, clean post-match handoff.

BGMS Companion shows a compact in-game status overlay and prepares a session summary for BGMS after the match ends.

Phase 1 scope

  • Match start and match end state
  • Phase, local health, weapon state
  • Kills, deaths, revives, killer notice
  • Roster-based alive player count

Policy guardrails

  • No real-time damage meter
  • No live location or minimap
  • No PUBG API calls from live GEP triggers
  • No service role or Supabase secrets in the app
